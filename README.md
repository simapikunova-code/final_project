# Анализ поведения пользователей Яндекс Афиши

Проект по исследовательскому анализу данных пользователей платформы Яндекс Афиша.

## Цель проекта
Выявить признаки, связанные с вероятностью повторных покупок и возврата пользователей.

## Используемые инструменты
- Python
- pandas
- numpy
- matplotlib
- seaborn
- SQLAlchemy
- PostgreSQL
- phik

## Содержимое проекта
- `final_project.ipynb` — основной ноутбук с анализом
- `requirements.txt` — зависимости проекта
- `.gitignore` — исключения для Git
- `.env.example` — пример переменных окружения

## Как запустить
1. Создать `.env` с параметрами подключения к БД
2. Установить зависимости:
   `pip install -r requirements.txt`
3. Запустить Jupyter Notebook
